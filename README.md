# swiper-jquery
基于jquery的轻量swiper
基本语法
  var swiper = new Swiper({
    container: '.swiper',
    loop: true,
    time: 300,
    autoPlay: 1000
  });
以实现功能 
  1.自定制外层父元素
    container
  2.边缘抵触
  3.循环播放
    loop: true
  4.自动播放
    autoPlay: 1000/'1000ms'
  5.自动计算outerWidth
  6.配置过渡需要时间
    time：100
